# UnitTestingPractice
### 3 types of Tests in Android 
 - Unit
 - instrumentation (Integration) 
 - UI

### Unit Testing benefits
 - confirm code work like a charm
 - simulate App End User
 - you don't need run real device or emulator to test cases
